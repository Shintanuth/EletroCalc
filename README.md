# ⚡ EletroCalc

Software desenvolvido em Python para auxiliar estudantes e profissionais de eletrotécnica.

## Funcionalidades

- Lei de Ohm
- Potência Elétrica
- Impedância
- Reatância Indutiva
- Reatância Capacitiva
- Associação Série
- Associação Paralelo

## Tecnologias

- Python 3
